# 企业级端到端电商数据分析项目 

## 📖 项目简介
本项目是一个企业真实场景（数据已脱敏）的端到端数据分析实战。涵盖了从底层原始数据的抽取、清洗，到中间层的数据库管理与客户画像提取，再到顶层交互式看板的设计与业务洞察输出。

---

## 🛠️ 技术栈与实施路径

### 1. 探索性数据清洗 (Python / Pandas)
**目标**：将非结构化的原始订单数据转化为可分析的洁净数据集。
* **数据清洗**：处理缺失值、重复项，并对异常评分数据进行平滑处理。

### 2. 客户画像与数据提取 (SQL)
**目标**：模拟企业数据库环境，通过 SQL 提取核心业务指标。
* **数据拉取**：编写多表联接（Joins）查询，整合客户、订单与物流信息。
* **画像提取**：通过聚合函数与窗口函数，提取客户消费频次、平均客单价及配送偏好。
* **业务逻辑查询**：筛选高价值客户（VIP）及分析不同配送方式下的评分差异。

### 3. 可视化建模与高精度 UI 设计 (Power BI)
**目标**：构建具备专业级交互体验的决策看板。
* **DAX 建模**：编写动态度量值，实现平均评分、动态金额格式化以及跨维度聚合计算。
* **高精度 UI 设计**：
    * 采用**像素级对齐**工艺，确保视觉元素分布严谨。
    * 统一**圆角卡片布局**，通过对比色与层级设计提升报表的高级感。
* **交互体验**：配置同步切片器与交互式直方图，支持用户自主进行下钻（Drill-down）分析。

---

## 💡 核心洞察
* **客户分层建议**：识别出特定高评分商品品类，建议针对该品类优化 Standard 配送效率以提升整体 ROI。
